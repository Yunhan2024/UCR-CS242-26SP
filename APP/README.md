# TMDB Movie Search Engine — CS242 Group 7

A search engine for 220,000+ movies from The Movie Database (TMDB).  
Supports **sparse search** (Elasticsearch) and **dense search** (BERT + FAISS), with an interactive world map visualization.

---

## Quick Start

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

### 2. Make sure Elasticsearch 8.x is running

```bash
curl http://localhost:9200
```

### 3. Build the Elasticsearch index (Part A)

```bash
./indexbuilder.sh /path/to/data.zip english_stem
```

> **Note:** The ES index must be rebuilt with `--recreate-index` to include the new `origin_country` field required for the world map.

### 4. Place BERT index files (Part B1)

Copy the two files generated by `CS242_BERT_Indexing.ipynb` into `backend/models/`:

```
backend/models/movie_faiss.index
backend/models/movie_metadata.pkl
```

### 5. Run the web application (Part B2)

```bash
cd backend
python app.py
```

Visit **http://localhost:5000** in your browser.

## Features

- **Dual-index search**: Toggle between Elasticsearch (BM25) and BERT + FAISS (semantic similarity)
- **Side-by-side comparison**: Compare ranking quality from both indexes for the same query
- **World map**: Choropleth visualization by origin country; click a country to filter results
- **Search timing**: Displays query execution time for performance comparison
- **External links**: Each result links to IMDb and TMDB pages
